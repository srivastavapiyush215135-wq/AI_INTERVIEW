# 🤖 AI Interview Platform

An AI-powered mock interview platform that helps users prepare for technical and HR interviews using Google Gemini AI.

## ✨ Features

- 📄 Resume Upload (PDF)
- 🤖 AI Resume Analysis
- ❓ AI-Generated Interview Questions
- 📝 Answer Evaluation
- 📊 Detailed Interview Report
- 📥 Download Interview Report (PDF)
- ⚡ FastAPI Backend
- 🎨 Streamlit Frontend

## 🛠️ Tech Stack

- Python
- FastAPI
- Streamlit
- Google Gemini AI
- SQLAlchemy
- SQLite
- PyPDF2
